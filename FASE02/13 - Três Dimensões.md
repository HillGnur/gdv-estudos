# Capítulo 13 – Três Dimensões: Introdução à Modelagem 3D

## 1. Panorama
O capítulo apresenta evolução histórica da modelagem 3D e aumento da demanda de mercado.

Destaques:
- Evolução de baixa para alta complexidade visual
- Crescimento de jogos 3D nas últimas décadas

## 2. Autodesk Maya
Maya é apresentado como ferramenta relevante na indústria.

Pontos abordados:
- Uso em grandes estúdios
- Ampla gama de ferramentas
- Acesso educacional para estudantes

## 3. Navegação no Maya
Conteúdo prático inclui:
- Contextos de trabalho (modeling, rigging, animation)
- Viewports
- Atalhos de navegação com Alt + mouse

## 4. Primitivos Poligonais
Base de modelagem com formas simples (cubo, esfera, cilindro etc.) para construir objetos mais complexos via box modeling.

## 5. Channel Box
Painel de propriedades com:
- Transformações (posição, rotação, escala)
- Subdivisões
- Controle de parâmetros do objeto

Atalhos citados:
- W, E, R

## 6. Edição de Componentes
Modos de edição:
- Vertex
- Edge
- Face

Atalhos citados:
- F9, F10, F11, F8

## 7. Ferramentas de Modelagem
### 7.1 Extrude
Cria nova geometria a partir de componentes selecionados.

### 7.2 Edge Loop Tool
Adiciona subdivisões localizadas para maior controle de forma.

### 7.3 Bevel
Arredonda arestas e melhora acabamento visual.

## 8. Boas Práticas
Recomendações:
- Pesquisar referências antes de modelar
- Organizar referências com PureRef
- Unir objetos com Combine
- Ajustar pivô com Center Pivot

## 9. Síntese
O capítulo reúne fundamentos técnicos de modelagem poligonal e fluxo inicial de trabalho no Maya.

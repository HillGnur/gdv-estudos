# Capítulo 12 – Animando na Unity

## 1. Contexto da Engine
Unity é apresentada como engine multiplataforma para desenvolvimento de jogos, com foco do capítulo em animação 2D e partículas.

## 2. Instalação e Configuração
Passos principais descritos:
- Acesso ao site da Unity
- Escolha da licença (incluindo Personal)
- Instalação via Unity Hub
- Instalação de editor e componentes adicionais

Também são citados requisitos mínimos de sistema.

## 3. Criação de Projeto
Fluxo inicial:
- Criar novo projeto no template 2D
- Definir nome e pasta
- Entender painéis principais (Assets, Scene, Hierarchy, Inspector)
- Importar assets

## 4. Animação com Sprite Sheet
Processo apresentado:
1. Importar sprite sheet
2. Configurar como Sprite/Multiple
3. Abrir Sprite Editor
4. Fazer Slice automático
5. Arrastar frames para cena
6. Criar clipe de animação
7. Editar timeline na aba Animation

## 5. Sistema de Partículas
Partículas são apresentadas como recurso para efeitos visuais (fogo, folhas, explosões etc.).

Configurações citadas:
- Velocidade inicial
- Rotação
- Gravidade
- Forma de emissão
- Quantidade emitida

Também são mencionados material, transparência e shader para resultado visual adequado.

## 6. Síntese
O capítulo funciona como guia prático de setup e primeiros passos para animação e efeitos na Unity.

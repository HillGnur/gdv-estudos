# Capítulo 14 – Boas Práticas: Modelagem 3D

## 1. Uso de Referências
O capítulo reforça uso constante de referência visual durante modelagem.

Ferramenta destacada:
- PureRef (organização e fixação de imagens de referência)

Funções citadas:
- Inserir imagens
- Organizar layout
- Salvar board
- Manter janela em primeiro plano

## 2. Exemplo Prático de Modelagem
Caso apresentado: mesa medieval com cadeiras.

Etapas resumidas:
- Construção da base com cubos
- Ajuste de forma com escala
- Acabamento com bevel
- Duplicação de peças
- Inserção de detalhes com edge loops e extrude
- União final de objetos e ajuste de pivô

## 3. Image Plane
Recurso para modelar sobre imagem de referência em viewport.

Fluxo descrito:
- Importar imagem na vista adequada
- Ajustar transparência/opacidade
- Usar imagem como guia de proporção e silhueta

## 4. Variações Rápidas de Asset
Estratégia:
- Criar modelo base
- Duplicar
- Alterar proporções e detalhes para gerar variações

Benefício:
- Ganho de tempo em produção de múltiplos objetos similares

## 5. Conexão de Objetos
Ferramenta destacada:
- Bridge para conectar faces/arestas entre malhas

Exemplo citado:
- Construção de alça de caneca

## 6. Refinamento
Recursos abordados:
- Bevel localizado para detalhamento
- Soften Edge para suavização visual

## 7. Boas Práticas Gerais
- Trabalhar sempre com referências
- Organizar e combinar objetos por etapas
- Conferir estrutura em wireframe
- Iterar comparando modelo com referência
- Praticar em assets simples antes de modelos complexos

## 8. Síntese
O capítulo consolida fluxo de trabalho prático para modelagem eficiente, com foco em referência, organização e refinamento progressivo.

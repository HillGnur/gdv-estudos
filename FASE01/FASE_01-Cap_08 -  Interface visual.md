# Interface Visual

## 1 Interface Gráfica de Usuários

### 1.1 Aspectos Gerais do Design de Interfaces

A interface gráfica é o elemento essencial que conecta o ser humano com máquinas como computadores, smartphones e diversos dispositivos tecnológicos. O termo **interface** refere-se ao elemento que proporciona uma ligação física ou lógica entre dois sistemas ou partes de um sistema que não poderiam estar conectados diretamente.

#### Histórico da Evolução

A noção de interface gráfica — manipulação de ícones por intermédio de um apontador (mouse) — foi popularizada com o **Apple Macintosh**. O objetivo era trazer ao grande público um sistema de manipulação de informações de fácil manuseio, fazendo analogias com objetos do dia a dia, como pastas, arquivos e lixeiras. O Macintosh instaura um diálogo entre homem e computador de forma quase orgânica.

**Cronologia das interfaces gráficas:**
- **1973**: Xerox PARC — primeira interface gráfica de usuário
- **1984**: Apple Macintosh — interface gráfica comercializada
- **Décadas de 1980 e 1990**: Windows (Microsoft) e Apple Macintosh dominam o mercado
- **Atualmente**: Smartphones, tablets, touchscreens e diversos dispositivos integram GUIs

#### Conceito de Interface Gráfica de Usuário (GUI)

A **Graphic User Interface (GUI)** consiste em todos os elementos do sistema com os quais as pessoas entram em contato, seja de forma física, perceptível ou conceitualmente. Em ambientes digitais, a interface é o elemento que separa dois espaços absolutamente diferentes (real e digital) e proporciona sua coexistência.

**Funções da interface gráfica:**
- Canal de comunicação do usuário final com o conteúdo de forma geral
- Mediação da interação das pessoas com diversos dispositivos tecnológicos
- Transição entre o mundo real e o digital
- Ambiente gráfico do produto digital

Vivemos em uma sociedade mediada pelas interfaces gráficas, utilizando-as em smartphones, tablets, notebooks, computadores, caixas eletrônicos, smart TVs e inúmeros novos dispositivos.

## 2 A Interação

### Formas de Interação com Sistemas

Os seres humanos interagem com os sistemas de muitas formas diferentes:
- Apertando botões e ícones
- Tocando em uma tela
- Movimentando um mouse
- Clicando no botão do mouse
- Manipulando barras de rolagem

### Perspectivas de Interação

#### Perspectiva da Percepção

As pessoas, em geral, interagem com um sistema por meio do que conseguem **ver**, **ouvir** e **tocar**. Os aspectos visuais do design de interface elaboram um design que permita às pessoas:
- Ver os elementos em uma tela
- Perceber a organização e hierarquia visual
- Interagir com os elementos, não importando o formato da tela

#### Perspectiva Conceitual

Conceitualmente, as pessoas interagem com sistemas e dispositivos por meio do conhecimento do que eles podem fazer e como podem fazer. É através das interfaces gráficas que os usuários manipulam diversos sistemas, como:
- Editores de texto
- Aplicativos de envio de mensagens instantâneas
- Jogos

### Modelo Mental e Navegação

De modo geral, as pessoas usam um **modelo mental** do que é o dispositivo e como ele funciona. Basta manipular algumas vezes esse dispositivo para que o mapa mental seja montado. Contudo, é necessário:

- Conhecer certos comandos existentes
- Entender como dados estão disponíveis e se apresentam
- Descobrir como chegar a determinadas informações (navegação)
- Encontrar detalhes de diversos elementos
- Ter uma visão geral dos elementos
- Focar em determinadas áreas

### Objetivo do Design de Interface

O design de interface trabalha para criar uma experiência que permitirá às pessoas utilizarem, da melhor forma possível, o sistema, que é o objeto do design (projeto).

## 3 WIMPs

**WIMP** é um acrônimo para:
- **W**indows (Janelas)
- **I**cons (Ícones)
- **M**enus (Menus)
- **P**ointers (Apontadores)

Estes são os principais pontos de interação do usuário com uma interface gráfica.

### Janelas

Uma janela é um meio de compartilhar os recursos de exibição gráfica de um dispositivo entre múltiplas aplicações ao mesmo tempo. 

**Características:**
- Permitem que a tela de uma estação de trabalho seja dividida em áreas
- Funcionam como canais independentes de entrada e saída de dados
- Podem ser colocadas sob o controle de diferentes aplicações
- Facilitam que as pessoas vejam a saída de vários processos simultaneamente
- Permitem que o usuário escolha qual janela receberá a entrada, selecionando-a com um dispositivo apontador (como mouse)

### Ícones

Um ícone é uma **imagem ou símbolo** usado para representar:
- Um arquivo
- Uma pasta
- Uma aplicação
- Um dispositivo (como uma impressora)

Os ícones fazem parte da manipulação direta do usuário e são usados para representar características e funções em aplicações de software, players de DVD, atalhos para configurações de sistema, entre outros.

**Vantagens dos ícones:**
- Ajudam as pessoas a reconhecer características que devem acessar
- É muito mais fácil lembrar-se do desenho de um ícone do que lembrar o comando textual
- Facilitam a identificação rápida de funções

### Menus

Um menu é uma **lista de comandos ou opções** entre os quais se pode escolher para realizar diversas ações.

**Características gerais:**
- Os comandos devem ser agrupados em tópicos
- Quando um item do menu é selecionado, uma ação é realizada
- Muito utilizados em websites para estruturar informação
- Proporcionam o principal método de navegação pelo conteúdo

#### Tipos de Menus

| Tipo | Descrição |
|------|-----------|
| **Menu Hierárquico (Cascata)** | Submenus se desdobram quando uma escolha é feita no menu de nível mais alto |
| **Menu Pop-up** | Não está ligado à barra de menu em um local fixo |
| **Menu Contextual** | Sua composição varia conforme o contexto no qual é acessado; oferece opções específicas do objeto selecionado |

**Utilização:**
- Websites: estruturação de informação
- Aplicativos para smartphones: geralmente na lateral esquerda

### Apontadores (Pointers)

O apontador é um dispositivo que controla a interação direta com objetos na tela. Exemplos incluem:
- **Mouse**: o mais disseminado
- **Dedo**: em sistemas touchscreen (mais populares atualmente)
- **Caneta stylus**: em smartphones e tablets
- **Joysticks**: bastante utilizados em controladores de jogos

#### Manipulação Direta

A forma de interação chamada **manipulação direta** significa que controlamos diretamente os objetos na tela em vez de usar comandos por meio de uma interface de linhas de comandos.

Com o advento das **superfícies multitoque**, tornou-se possível o reconhecimento de uma ampla gama de gestos além de uma simples operação de apontar e selecionar. As grandes empresas de tecnologia registram patentes para proteger gestos ligados diretamente ao projeto desenvolvido, prevenindo que competidores copiem e vendam produtos com o mesmo gesto.

## 4 Outros Elementos Presentes nas Interfaces Gráficas

Além dos quatro principais pontos de interação WIMP, outros elementos ajudam a compor os pontos de interação de uma interface gráfica de usuário:

| Elemento | Função |
|----------|--------|
| **Botões de Rádio** | Permitem que as pessoas façam escolhas exclusivas |
| **Caixas de Verificação** | Usadas para exibir configurações individuais que podem ser marcadas e desmarcadas |
| **Barras de Ferramentas** | Uma coleção de botões agrupados de acordo com a função de cada item |
| **Caixas de Listagem** | Oferecem diferentes maneiras de visualizar o conteúdo (listas com mais ou menos detalhes, ícones, miniaturas) |
| **Controles Deslizantes** | Um widget que pode retornar valores análogos; ideais para ajuste de volume, brilho, seleção de cor ou rolagem de documentos |

## 5 Recursos para Prática e Desenvolvimento

### GIMP - Ferramenta de Edição de Imagens

Para projetar e desenhar interfaces visuais, é recomendável conhecer e manipular softwares gráficos como o **GIMP** (gratuito):
- Download: https://www.gimp.org
- Tutoriais em inglês: https://www.gimp.org/tutorials/
- Tutoriais em português: http://www.gimpbrasil.org/tutoriais

### Bancos de Assets para Prototipação

Construir um banco de assets (sprites, tilesets, texturas, fundos e elementos de UI) é essencial para ampliar o repertório visual. Recomenda-se:

| Recurso | Tipo de Asset | Licença |
|---------|---------------|---------|
| **Freepik** | Ilustrações, vetores, recursos visuais | Gratuitos com atribuição ou pagos |
| **GameArt2D** | Sprites, tilesets, packs 2D | Freebies em CC0; pagos em royalty-free |
| **Kenney** | Assets variados (2D, áudios, etc.) | CC0 (domínio público) |

**Boas práticas na organização de assets:**
- Seleção baseada em coerência de estilo
- Avaliação de legibilidade e escala
- Adequação ao gênero do jogo
- Registro de origem e licença de uso
- Inclusão de exigências de atribuição quando aplicável

### Pinterest - Repositório de Referências

O Pinterest é um excelente repositório de ideias, referências e trabalhos diversos. Recomenda-se:
- Acessar: https://www.pinterest.com
- Realizar buscas com palavras-chave: "design", "interface" e "inspiração"
- Montar pastas com os melhores trabalhos encontrados
